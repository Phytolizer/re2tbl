DFA structures¶

namespace re2tbl¶

All of the re2tbl declarations are within this namespace.

struct Dfa¶
#include <dfa.hpp>

A deterministic finite automaton (DFA).

Deterministic means that given a current state (node) and a current input character, the machine can confidently make a single unambiguous move, or report failure.

Unlike the NFA, the DFA never contains epsilon moves, which is the reason the above statement holds.
