NFA structures¶

namespace re2tbl

All of the re2tbl declarations are within this namespace.

struct Nfa¶
#include <nfa.hpp>

A non-deterministic finite automaton (NFA).

Contains a set of nodes, as well as a pointer to its start node.

Public Functions

NfaNode *AddNode()¶

Create a new node within the NFA.

It will initially be nonaccepting and have no edges leading out.

You will probably not call this function directly, as it is called by ParseRegex().

void ConnectNodes(NfaNode *a, NfaNode *b, char transition = '\0')¶

Create an edge between two nodes. The edge will require consuming a character to move if transition is not the null character.

Like AddNode(), it is called by ParseRegex() and calling it directly is unnecessary.

Parameters

  • a – the source node

  • b – the destination node

  • transition – the character that is consumed by the new edge, or '\0'.

void Display(std::ostream &os) const¶

Display a human-readable representation of this NFA.

Describes each node by its ID and the IDs that it “points” to via its edges.

Additionally, the representation specifies the accepting node(s).

Parameters

os – the stream to output to

Public Members

std::vector<std::unique_ptr<NfaNode>> nodes¶

The nodes which make up this NFA.

Note that you should not iterate these in order, as it is meaningless. Use each NfaNode’s edges to make moves based on the input string.

NfaNode *start¶

A pointer within nodes that refers to the initial state.

In practice, this is always equal to &nodes[0].

struct NfaEdge¶
#include <nfa.hpp>

An edge between NFA nodes.

Can represent an epsilon move or a move on one or more chars.

Public Members

bool epsilon¶

Whether this edge can be traversed “for free” (without consuming a character).

  • true: no character is consumed

  • false: a character is consumed

std::bitset<std::numeric_limits<char>::max() + 1> chars¶

The set of characters that need to be consumed to traverse this edge.

It will be empty if epsilon is true.

NfaNode *next¶

The other end of this edge.

When traversing, this pointer is the destination.

struct NfaNode¶
#include <nfa.hpp>

A single node of the Nfa structure.

Maintains its own index as id for convenience.

Public Members

std::size_t id¶

The index of this node in the Nfa, used as a unique identifier.

bool accepting¶

Whether the NFA can “accept” on this node.

Accepting means that if the input is exhausted in this state, the machine reports success.

std::vector<NfaEdge> next¶

The edges which lead out from this node, if any.
