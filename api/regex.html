Regular expression parsing¶

namespace re2tbl

All of the re2tbl declarations are within this namespace.

Enums

enum RegexTokenType¶

Classifies a regular expression component for parsing.

Primarily used as a member for RegexToken.

Values:

enumerator kLiteral¶

A literal. This is the “catch-all” category, and includes any character not specially handled.

enumerator kLeftParen¶

A left parenthesis; begins a grouping to override the precedence rules.

enumerator kRightParen¶

A right parenthesis; closes a grouping.

enumerator kPipe¶

A pipe character. Represents multiple alternatives that are valid in a particular regular expression.

enumerator kStar¶

A star character. When placed after a subexpression, that subexpression can occur any number of times, including zero.

enumerator kPlus¶

A plus character. When placed after a subexpression, that subexpression can occur any number of times, but must occur at least once.

enumerator kEof¶

The end of the expression, used for convenience in the parser implementation.
